## Install

The easiest way to install the module is by using [modman](https://github.com/karlssonlord/modman):

1. `modman clone git@github.com:karlssonlord/KL_LessFriction.git`
2. `modman deploy KL_LessFriction`
3. Clear the cache
4. Check out
